== Description ==

View your php arrays in an slide-out off canvas menu. Created specifally for WordPress theme development and Advanced Custom Fields.

== Important Notes ==

//standard (acf)
prez(); // view advanced custom field of current template

//post ID (acf)
prez(3); // view advanced custom field data of specific page id

//general array (non-acf)
prez($array); // view a non ACF array
